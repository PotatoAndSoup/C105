Leer la predicción

Descripción : Usa la síntesis de voz para leer el resultado de la predicción

  1. Definir una variable para almacenar la API de voz
  2. Almacena la etiqueta del resultado de la predicción que queremos que el sistema hable en una variable
  3. Usa la función SpeechSynthesisUtterance()para convertir el texto en voz, pasa la variable que mantiene el resultado de la predicción a SpeechSynthesisUtterance() como parámetro y almacénalo en una variable
  4. Después usa la función speak() y pasa la variable que sostiene el texto a los datos de discurso convertidos a la función speak()
  5. Escribe todo el código anterior dentro de la función gotResult() en sketch.js
  6. Para referencia - Para recordar la API de voz, consulta el código JS de la clase 100 (aplicación Toma mi selfie)
